virtual Observation & MLBridge::Environment::step ( Action action)
pure virtual

Step function takes an action as input and returns the observation corresponding to the next state. This method should be implemented by the user. Typically this method should call setDone() if the termination condition is met. setNextAgent() can be called in this method to set the next agent to interact with.
