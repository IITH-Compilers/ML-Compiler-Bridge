9// This file defines the C APIs for ONNXModelRunner.
10// This is a wrapper around the ONNXModelRunner class that provides an interface
11// for the MLCompilerBridge to interact with the ONNX models during inference.
12//
13// Usage for single agent:
14// 1. Create an ONNXModelRunner object using createSingleAgentOMR
15// 2. Evaluate the features using singleAgentEvaluate
16// 3. Destroy the instance of ONNXModelRunner using destroyONNXModelRunner
17//
18// Usage for multiple agents:
19// 1. Create an Environment object using createEnvironment
20// 2. Set the number of features using env_setNumFeatures
21// 3. Set the step function using env_setStepFunc
22// 4. Set the reset function using env_setResetFunc
23// 5. Set the next agent using env_setNextAgent
24// 6. Create an ONNXModelRunner object using createONNXModelRunner
25// 7. Evaluate the features using evaluate
26// 8. Destroy the instance of ONNXModelRunner using destroyONNXModelRunner
27// 9. Destroy the instance of Environment using destroyEnvironment
28//
29// Using Environment:
30// 1. Create an Environment object using createEnvironment
31// 2. Set the number of features using env_setNumFeatures
32// 3. Set the step function using env_setStepFunc
33// 4. Set the reset function using env_setResetFunc
34// 5. Set the next agent using env_setNextAgent
35// 6. Destroy the instance of Environment using destroyEnvironment after calling
36// destroyONNXModelRunner.
37//
38// Internally, the ONNXModelRunner will call the step function to get the next
39// action and the reset function to reset the environment. The step function
40// should return a pointer to an array of floats. The reset function should
41// return a pointer to an array of floats.
42//
43//===---------------------------------------------------------------------===//
