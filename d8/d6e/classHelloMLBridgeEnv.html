Reset function returns the initial observation. This method should be implemented by the user. This method can internally call setNextAgent() to set the next agent to interact with and setDone()/resetDone() to set/reset the termination condition.

Step function takes an action as input and returns the observation corresponding to the next state. This method should be implemented by the user.
